# Bat_signal
##  what is this??
You might wonder what is this, this is blinky in the shape of bat. it symbolise the truth is always brightter than the lie. This isnt meant to be just another hardware project but also an art piece which u can carry everywhere with you. it can also  be used as flashlight.

## why bat signal??
this is blinky with 10 leds in middle of bat in circle like bat signal in night of gotham. it uses an ic to command the leds to light up and has built-in clock to set fixed interval of time before the led shines again

## assembly
1) dowload the gerber file the repo
2) upload it to jlcpcb
3) chose the white pcb (as colourfull silkscreen only works on white pcb)
4) set surface finsih to enig
5) in advance setting, select easyeda colourful silkscreen
6) order the pcb and components
7) solder the componets
